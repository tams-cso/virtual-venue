hackTAMS Virtual Venue

Welcome to the virtual venue! This application was designed to replace the traditional hackathon venue in-person that we cannot provide this year. This web app uses Discord to both authenticate and provide the voice channels. Upon logging in, you will be prompted to join the server, if you are not already in the server. Please join the main vc under the game channel category to be able to connect to the voice channels. The controls are WASD or Arrow keys to move around your character. Have fun and good luck with your hacking!
